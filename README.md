### 요구사항

- 선착순으로 강의에 신청
  - 동시성을 고려해야함.
- 동일한 신청자는 한 강의에만 수강신청 가능
- 강의의 최대 신청 인원이 꽉 차게되면 신청이 불가능해야함.

---

- 특강 신청 완료 여부 조회를 했을 때 성공했다면 신청 완료, 실패했다면 신청 실패를 반환해야 함.
